安装 Composer

安装 Composer

执行下面的命令，在 Linux / Unix / macOS 下安装 Composer ，并且可以在系统中全局调用。

curl -sS https://getcomposer.org/installer | php
sudo mv composer.phar /usr/local/bin/composer

在 Windows 下安装只需下载并运行 安装程序 ，它将安装最新版本的 Composer ，并设置好系统的环境变量。
