# 配置文件

要开始在你的项目中使用 Composer ，只需要一个 `composer.json` 文件。

```json
{
    "name": "kuriv/manual-composer",
    "description": "Composer 手册",
    "type": "project",
    "license": "MIT",
    "authors": [
        {
            "name": "Kuriv Vesti",
            "email": "kuri.personal@gmail.com"
        }
    ],
    "minimum-stability": "stable",
    "require": {}
}
```
